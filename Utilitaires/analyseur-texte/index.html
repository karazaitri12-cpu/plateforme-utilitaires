🔢 Comment sont comptés les éléments ?

Comptage des mots :

  • Supprime les espaces en début et fin de texte
  • Divise sur tous types d'espaces (espaces, tabulations, retours)
  • Filtre les chaînes vides pour un comptage précis

Comptage des caractères :

  • Compte tous les caractères sans exception
  • Inclut lettres, chiffres, espaces, ponctuations, accents

Comptage des phrases :

  • Divise sur les terminateurs : . ! ?
  • Gère les ponctuations multiples ("!!!", "...")
  • Compte uniquement les phrases avec du contenu

📁 Formats de fichiers supportés

  • .txt : Texte brut
  • .docx : Word moderne (via Mammoth.js)
  • .pdf : PDF texte (via PDF.js)

💡 Conseils d'utilisation

  • Analyse en temps réel pendant la saisie
  • Compatible avec textes longs (plusieurs milliers de mots)
  • Usage médical : parfait pour articles, rapports, contenus web
