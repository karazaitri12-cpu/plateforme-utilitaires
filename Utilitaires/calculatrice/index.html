Mode d'emploi

Opérations de base
Exemple : 10 + 5
Tapez 1 0 + 5 puis =.

Pourcentage contextuel : 200 + 10% donne 220.

Mémoire (MC/MS/M+/M-)

Utilisez MS pour stocker une valeur, M+ pour additionner à la mémoire, et MR pour la rappeler.

Ex : 50 → MS → 20 → M+ → MR affiche 70.
Fonctions Scientifiques

Cliquez sur Trigo/Sci pour afficher les fonctions avancées (sin, cos, log, √, etc.).

Pour les puissances, utilisez le bouton x^y (ex: 2 x^y 3 = 8).

pH approximatif : log(0.001) → -3 (pH ≈ 3)
